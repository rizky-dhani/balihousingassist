## MODIFIED Requirements

### Requirement: Public Property Grid
The system SHALL display all available properties in a responsive grid layout on the homepage/catalog page using reactive Livewire components, prioritizing properties marked as "Featured" in the default view.

#### Scenario: View all properties with featured prioritization
- **WHEN** a visitor accesses the website
- **THEN** they see a list of available properties with their names, types, and starting prices
- **AND** properties marked as "Featured" MUST be displayed at the top of the list, followed by non-featured properties
